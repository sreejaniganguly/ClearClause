# CLEARCLAUSE

ClearClause is an open-source, AI-powered legal document analyzer that runs entirely in your browser. Upload a PDF, get an instant clause-by-clause breakdown in plain English — with risk flags, negotiation suggestions, and references to relevant Indian laws.

WHAT IT DOES:

1. Breaks down every clause with a Safe / Review / Danger risk rating
2. Suggests fairer alternative wording for risky clauses
3. Cites Indian laws being violated for dangerous clauses
4. Shows an overall risk score and summary
5. Supports analysis output in English, Hindi, and Bengali
6. Lets you ask questions about the document in plain English
7. Works with PDF and image uploads
8. Your API key stays in your browser — nothing is stored on any server

SETUP:
The app is already deployed and hosted using Vercel. No need to install anything locally.

API KEYS:
ClearClause uses OpenRouter API to call AI models
Keys are saved in your browser's localStorage only. Never hardcoded.
